\doxysubsection{Descripción detallada}
Representa el conjunt de característiques y operacions de les especies. 

Ofereix la posibilitat de calcular els k-\/meros d\textquotesingle{}una especie. A partir de dues especies podem calcular la distància que les separa genèticament.

Donat que hem d\textquotesingle{}establir un paramètre k, tenim la necessitat de crear una constructora d\textquotesingle{}especie buida per poder introduir la k.
