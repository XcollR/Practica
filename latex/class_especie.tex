\doxysubsection{Descripció Detallada}
Representa el conjunt de característiques y operacions de les especies. 

Ofereix la posibilitat de calcular la distancie entre dues especies, consultar el gen, i escriure el gen. Donat que hem d\textquotesingle{}establir un paramètre k, tenim la necessitat de crear una constructora d\textquotesingle{}especie buida per poder introduir la k.
