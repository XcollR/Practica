\doxysubsection{Descripció Detallada}
Representa un conjunt de clusters. 

Ofereix la possibilitat d\textquotesingle{}ajuntar clusters per tal d\textquotesingle{}anar creant l\textquotesingle{}arbre filogenètic. També pots demanar imprimir un cluster determinat si existeix i imprimir l\textquotesingle{}arbre resultant. També ofereix una taula de distancies entre clusters. Els clusters es generen mitjançant la sincronització amb un conjunt d\textquotesingle{}especies i es compleix amb la funció inicializa\+\_\+clusters.
