Representa un cluster d\textquotesingle{}un arbre binari. 

Ofereix les opcions d\textquotesingle{}imprimir l\textquotesingle{}arbre del cluster amb les distancies respecte als seus fills. També l\textquotesingle{}opció de crear un cluster a partir de la fusió de dos altres.
